# cpp-dsa-bank-backend
C++ Console Banking System + DSA Practice. Built by Senior Backend Dev, 10yr exp.

## Features
1. User register + login with password check
2. Create bank account, deposit, withdraw, check balance  
3. File I/O: Saves all data to data.txt automatically
4. Memory safe: Destructor cleans up all objects
5. OOP design: User, Account, Bank classes

## C++ Concepts Shown
Classes, Objects, Encapsulation, vector, File I/O, Constructors/Destructors, Pointers

## How to Compile & Run
g++ *.cpp -o bank && ./bank
